Тестирование маршрутов:

1. Зарегестрированному пользователю доступны: главная страница, список своих записей,
страница отдельной записи, логин и регистрация. +
2. Анонимному пользователю доступны главная страница, логин и регистрация. +
3. Редактирование, удаление и просмотр записи доступны только автору. +
4. При попытке перейти на страницу редактирования или удаления записи
анонимный пользователь перенаправляется на страницу авторизации. +
5. Авторизованный пользователь не может зайти на страницу редактирования 
или удаления чужих записей (возвращается ошибка 404). +


Тестирование контента:

1. Записи отсортированы по порядку. +
2. Анонимному пользователю не видна форма для создания записи, а авторизованному видна. +
3. Редирект на успешную форму. +


Тестирование логики:

1. Анонимный пользователь не может оставлять записи. +
2. Авторизованный пользователь может оставлять записи. +
3. Авторизованный пользователь может редактировать и удалять свои записи. +
4. Авторизованный пользователь не может редактировать и удалять чужие записи. +
